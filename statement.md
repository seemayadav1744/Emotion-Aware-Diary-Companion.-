# Problem Statement

Students face emotional stress, anxiety, loneliness, and academic pressure. Yet, there is no offline, simple system that understands their mood and provides instant emotional support. This project aims to create a Java-based emotion-aware companion that responds with motivation and helpful suggestions based on the user's emotional state.

---

# Scope of the Project

The project focuses on:
- Detecting and interpreting user emotion
- Providing personalized motivational responses
- Storing emotional history for self-reflection
- Offering a clean, simple console interface

This system is lightweight, offline, and fully implemented in Java using OOP principles.

---

# Target Users
- Students facing emotional or academic stress  
- Individuals who prefer offline, private emotional support  
- Anyone who wants a simple emotional tracking tool Bmylr 

---

# High-Level Features
- Emotion detection (happy, sad, stressed, angry, lonely, anxious)
- Emotion-based motivational support
- Activity suggestions (music, rest, meditation, fresh air)
- Mood history tracking and retrieval  
- OOP-based modular architecture  
- File-based local storage for privacy
